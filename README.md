# flash_card


## Getting Started

- Install package:
```
    dependencies:
    flash_card:
        git:
            url: https://github.com/tranhieudev/flash_card.git
```


- Example:
```
   FlashCard(
       frontWidget: Container(
          child: Text(
               'front'
          )
       ),
       backWidget: Container(
           child: Text(
                'back'
           )
       )
   )
```
