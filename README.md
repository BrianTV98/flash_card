## flash_card


## Include to your project

In your `pubspec.yaml` root add:

```yaml
dependencies:
  flash_card: ^0.0.5
```

![Showscase gif](https://github.com/BrianTV98/flash_card/blob/main/assets/flash_card_demo.gif)

```
   FlashCard(
       frontWidget: Container(
          child: Text(
               'front'
          )
       ),
       backWidget: Container(
           child: Text(
                'back'
           )
       )
   )
```

```
   FlashCardListItem(
       flashcards: []
   )

```
